## Key Learnings

1. **Smart mode finds data integrity anomalies well** — extreme outliers surface naturally via find_anomalies
2. **Manual+skills finds operational patterns better** — skills guide agent to check temporal, segment shifts, payment patterns
3. **Temporal gap in smart mode** — detect_pattern didn't auto-add temporal steps (fixed post-r1)
