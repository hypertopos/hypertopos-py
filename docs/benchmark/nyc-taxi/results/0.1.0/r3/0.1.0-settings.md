## Key Learnings

1. **Smart finds zone structure well** — ghost zones, airports, Staten Island, Manhattan volume all surfaced through geometric patterns
2. **Manual finds rate code/fare encoding issues** — JFK misuse (12,768 trips), fare corruption ($1.95), rate code 99 — require domain-specific investigation recipes from skills
3. **Temporal: smart 2/3, manual 3/3** — smart found regime change + drift + trajectories but missed New Year's Day. Manual found New Year's Day (26% demand drop) because skills guide explicit temporal comparison
4. **Both find cross-validation** — regime change Feb 19 found by both modes independently
5. **Complementary** — A finds 15, B finds 12, overlap on 5 findings, combined = 22 unique
