## Key Learnings

1. **Smart mode A improved +5 pts (67%->79%)** from server-side intelligence alone — no instruction change
2. **Smart finds different things than manual** — A finds data integrity (billing, GPS, timestamps), B finds operational patterns (holidays, geographic structure, rate codes)
3. **Combined A+B = 22 unique findings** — neither mode is complete alone, Exp C (production) should combine both
4. **Temporal remains smart mode ceiling (1/3)** — keyword fallback cannot parse dates or understand "New Year's Day". Auto compare_time_windows (baseline vs recent) added post-r2 for r3+
5. **Manual+skills is more expensive** (~200K+ vs ~91K tokens) but more thorough on temporal and operational patterns
6. **Skills update (post-r1) improved B by +1 pt** — exhaustive enumeration, mandatory clusters, temporal fallthrough rules
